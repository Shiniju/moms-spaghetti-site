# Promptai PRO Studio

React + Vite로 만든 이미지 생성 프롬프트 스튜디오입니다.

## 기능

- 컨셉 문장을 구조화된 이미지 프롬프트로 변환
- Midjourney, Stable Diffusion, Normal 프롬프트 출력
- JSON 스키마 복사 및 다운로드
- 카메라 감성, 화풍, 화면비 프리셋 지원
- GitHub Pages 자동 배포

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

GitHub Actions가 `main` 브랜치 push 시 자동으로 `dist` 결과물을 GitHub Pages에 배포합니다.
